# cypress-testing

✍🏼 This demonstrates how to write BDD tests using Cypress for automation.

🏃‍♀️ Run these command after you clone the repo
 ```javascript
npm i
```
 ```javascript
npx cypress open
```
Choose the feature you want to run test
